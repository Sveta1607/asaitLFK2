# Telegram-бот записи (демо)

Бот на **Node.js** + **Telegraf**: выбор специалиста → день → время → имя и фамилия → итог. **Специалисты, слоты и записи** берутся из той же **PostgreSQL**, что и сайт, через FastAPI (`/api/telegram/...` с секретом). Сессии диалога по-прежнему в памяти процесса.

## Запуск

1. Установите [Node.js](https://nodejs.org/) (LTS).
2. Запустите **бэкенд** (`backend`) с настроенным `DATABASE_URL` и добавьте в `backend/.env` секрет (одинаковый с ботом):

   ```env
   TELEGRAM_BOT_API_SECRET=ваша_длинная_случайная_строка
   ```

3. Создайте бота в Telegram через [@BotFather](https://t.me/BotFather), скопируйте токен.
4. В папке `telegram-bot` создайте файл `.env` (можно скопировать из `.env.example`):

   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
   TELEGRAM_BOT_API_SECRET=та_же_строка_что_в_backend_.env
   API_BASE_URL=http://127.0.0.1:3000
   ```

   Если API на другом хосте/порту — укажите полный URL (как у деплоя сайта).

4. Запуск:
   - **Через двойной щелчок:** откройте папку `telegram-bot` (её можно держать на рабочем столе) и дважды щёлкните **`start-bot.bat`**. При первом запуске скрипт сам выполнит `npm install`, если нет `node_modules`.
   - **Ярлык на рабочем столе:** один раз запустите **`Создать ярлык на рабочем столе.bat`** — появится ярлык «Запуск бота ЛФК»; дальше бот запускается с рабочего стола двойным щелчком по ярлыку.
   - **Из терминала:** `cd telegram-bot` → `npm install` → `npm start`.

5. В Telegram откройте своего бота и отправьте `/start`.

Команды: `/start` — новая запись, `/cancel` — сброс сценария.

Остановка: в консоли `Ctrl+C`.

## Разработка

```bash
npm run dev
```

Перезапуск при изменении файлов (Node `--watch`).
